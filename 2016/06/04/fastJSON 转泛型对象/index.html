fastJSON 转泛型对象

fastJSON在实际的使用过程中，需要将JSON字符串转成带有泛型的对象。普通的方式无法实现。查询相关的资料最后得到的结果如下：

public List<ResInnoIds> callStatByInno(List<Integer> innoIds){
    Map<String,Object> request = new HashMap<>();
    request.put("ccId",inputParamHead.getCcId());
    request.put("vdnId",inputParamHead.getVdnId());
    if(innoIds!=null&&innoIds.isEmpty()){
        request.put("innoIds",innoIds);
    }
    String url = CmsEndPoint.get("real.accesscode.callstatbyinno");
    String response = HttpUtil.post(url,request);
    Response<List<ResInnoIds>> res= JSON.parseObject(response,new TypeReference<Response<List<ResInnoIds>>>(){});
    return res.getResult();
}

由于Response<List<ResInnoIds>>是一个泛型对象。
使用JSON.toJavaObject(json,Response.class）无法将其转成Response>。
必须使用

JSON.parseObject(response,new TypeReference<Response<List<ResInnoIds>>>(){});
